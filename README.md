## Run Locally

Clone the project

```bash
  git clone https://github.com/abhaypsamrat/Quiz-Application.git
```

Go to the project directory

```bash
  cd Quiz-Application
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
